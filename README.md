# 🌐 Network Design & Security Projects (Packet Tracer)

This repository contains network design and network security projects created using Cisco Packet Tracer as part of university coursework.

---

## 📌 Project Description

These projects demonstrate practical implementation of networking concepts, including network design, configuration, and basic security practices.

The work was completed as part of networking-related courses, focusing on building and securing computer networks.

---

## 📂 Contents

* 📁 **Network Design Project**
  A complete network topology including routers, switches, and end devices with proper configuration.

* 🔐 **Network Security Project**
  A secured network implementation that includes security measures such as access control and network segmentation.

---

## 🛠️ Tools & Technologies

* Cisco Packet Tracer (.pkt files)
* Networking Concepts (Routing, Switching)
* Basic Network Security Practices

---

## 🚀 Features

* 🌐 Design of complete network topologies
* 🔌 Device configuration (routers & switches)
* 🔒 Implementation of basic security concepts
* 🧪 Simulation and testing using Packet Tracer

---

## ⚙️ How to Use

1. Install Cisco Packet Tracer
2. Open the `.pkt` files using the program
3. Explore the network topology and configurations

---

## 🎓 Academic Purpose

These projects were developed as part of university courses in Networking and Network Security to apply theoretical knowledge in practical scenarios.
